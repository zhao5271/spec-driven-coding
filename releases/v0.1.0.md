# v0.1.0

Initial public release of the `spec-driven-coding` bundle.

## Highlights

- synced the published repository with the latest local `spec-driven-coding` workflow updates
- added the full change lifecycle script set for planning, approval, status updates, validation, close-out, and catch-up
- added reusable change templates and task examples for common implementation shapes
- rewrote the README for end users, with clearer install paths, first prompts, workflow cheatsheet, and FAQ
- added a short public install and release note for users who want to install or pin the skill

## Install

Main skill only:

```bash
npx skills add zhao5271/spec-driven-coding -g -y
```

Full bundle:

```bash
git clone git@github.com:zhao5271/spec-driven-coding.git
cd spec-driven-coding
./install.sh
```
