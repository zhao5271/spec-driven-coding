# Skill Decision Table

Use this as the fast lookup version.

| Work slice | Skill |
| --- | --- |
| Page, component, styling, interaction | `frontend-design` |
| Backend API route or contract | `api-design-principles` |
| PostgreSQL schema or index work | `postgresql-table-design` |
| New implementation after planning | `test-driven-development` |
| Bug investigation or flaky behavior | `systematic-debugging` |
| Final completion gate | `verification-before-completion` |
| Risky or wide-scope review | `requesting-code-review` |
| Mixed UI + API + DB change | keep planning here, then combine matching skills |
